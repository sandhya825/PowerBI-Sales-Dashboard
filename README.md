# PowerBI-Sales-Dashboard
Developed an interactive sales analytics dashboard in Power BI to monitor business performance and support data-driven decision-making. The dashboard provides insights into sales trends, top-performing products, customer behavior, and regional performance through dynamic visualizations and KPI tracking. 
# 📊 Sales Dashboard using Power BI

# 📌 Project Overview

This project presents an interactive Sales Dashboard built using Power BI to analyze business performance and support data-driven decision-making. The dashboard provides insights into sales trends, product performance, customer behavior, and regional sales distribution.

# 🎯 Objectives

- Monitor key sales metrics and business KPIs.
- Identify top-performing products and regions.
- Analyze customer purchasing behavior.
- Track revenue trends over time.
- Enable data-driven business decisions.

# 🛠️ Technologies Used

- Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Microsoft Excel / CSV
- Data Visualization

# 📂 Dataset

The dataset contains sales transaction information including:

- Order Date
- Product Category
- Product Name
- Sales Amount
- Quantity Sold
- Customer Details
- Region Information

# 📈 Dashboard Features

- Revenue Analysis
- Sales Trend Analysis
- Regional Performance Analysis
- Product Performance Dashboard
- KPI Cards
- Interactive Filters and Slicers
- Customer Insights

# 🔍 Key Insights

- Identified top-performing products contributing to overall revenue.
- Analyzed monthly and yearly sales growth trends.
- Compared sales performance across regions.
- Discovered customer purchasing patterns.
- Highlighted opportunities for business growth.
